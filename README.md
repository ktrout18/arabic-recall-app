# Arabic Recall — hosted app (public mirror)

Installable PWA build of a personal Arabic drilling app. This repo holds the
**built site only**, published so GitHub Pages can serve it over HTTPS.

- App: https://ktrout18.github.io/arabic-recall-app/
- Drill prompts carry short source citations for provenance; the underlying
  course PDFs and their extracted text are **not** published here.
- Source code and content pipeline live in a private repository.
- Do not edit this repo by hand — it is overwritten on every publish.
